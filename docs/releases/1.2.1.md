## v1.2.1

![Codex EchoInk v1.2.1 发布图 / Release poster](https://raw.githubusercontent.com/AKin-lvyifang/codex-echoink/main/docs/images/codex-echoink-v1.2.0-release.jpg)

### 中文

**轻量知识库命令菜单：** 在知识库频道输入 `/` 后，命令菜单现在更接近 Codex 的轻量交互：默认透明、文字中性，只在当前项显示浅灰选中状态。

![EchoInk v1.2.1 轻量知识库命令菜单 / Lightweight Knowledge command menu](https://raw.githubusercontent.com/AKin-lvyifang/codex-echoink/main/docs/images/codex-echoink-v1.2.1-command-menu.png)

#### 更新内容

- 移除厚重的背景卡片和粉色命令文字，改用黑色命令名、灰色图标与简短说明。
- 支持 `↑` / `↓` 循环选择命令，长列表会自动滚动到当前项。
- `Enter` 只把当前命令填入输入框，不自动发送；`Esc` 关闭菜单，减少误触发。

#### 使用方法

1. 安装 `v1.2.1` 并重载 Obsidian。
2. 进入知识库频道，在输入框中输入 `/`。
3. 用 `↑` / `↓` 选择命令，按 `Enter` 填入后再补充参数或确认发送。

#### 下载

Obsidian 官方安装会读取本 Release 下的单文件资产：`main.js`、`manifest.json`、`styles.css`。

### English

**Lightweight Knowledge command menu:** Typing `/` in the Knowledge channel now opens a cleaner Codex-style list: transparent by default, neutral in color, and highlighted only for the active command.

The screenshot above shows the updated menu in a real Obsidian workspace.

#### What's New

- Replaced heavy background cards and pink command text with black command names, neutral-gray icons, and concise descriptions.
- Added wraparound `ArrowUp` / `ArrowDown` navigation with automatic scrolling for long lists.
- `Enter` fills the selected command without sending it; `Escape` closes the menu to reduce accidental runs.

#### How to Use

1. Install `v1.2.1` and reload Obsidian.
2. Open the Knowledge channel and type `/` in the composer.
3. Select a command with `ArrowUp` / `ArrowDown`, then press `Enter` to fill it before adding parameters or sending.

#### Download

Obsidian reads these single-file Release assets directly: `main.js`, `manifest.json`, and `styles.css`.
